Luna versions

Luna admin pakcage contents.

LUNA Admin Theme is a premium admin theme with flat and dark design concept. It is fully responsive admin theme built with latest Bootstrap 3+ Framework, HTML5 and CSS3, Media query. It can be used for monitoring or administration web applications, project management system, admin dashboard, application backend or other custom project.

We are release continuous long term updates and many new features will be coming soon in the near future updates. Once you purchased LUNA, you are entitled to free download of all updates.

LUNA theme includes two versions. One with all static html/css/js files that is great for any php, ruby or other programing language application and one special version for Meteor for node.js developers. In package you will also find SeedProject It is an application skeleton for a typical web app. You can use it to quickly bootstrap your web app projects.

HTML/CSS/JS version

HTML/CSS/JS version
+ Seed Project

This version contains all components, views and plugin created with pure HTML and Javascript code. This version is great if you are a PHP, Ruby, JAVA programmer - or any other language as each page is created separately it allows you to quickly implement it for your project by including needed resources. The html code, file structure and js code are is very clean and modular so you should not have any problems with its implementation. Additional we include full LESS files to easily change colors of theme.

METEOR Version

METEOR version
+ Seed Project

This version contains Meteor project (METEOR@1.4.3.1) with all designs, views and components from Luna theme. Project is created in accordance with the convention of Meteor structure and Blaze templating. As an router we it use FlowRouter. (Great for for discover Meteor framework.)

AngularJS Version

AngularJS version
+ Seed Project

Angular version is built with the latest technology (AngularJS 1.6.3) contains well organized and coded controllers and directives written specifically to support angular framework. It is provided with full Grunt/Bower environment.
